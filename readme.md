# Projeto Chat

Este é projeto idealizado com objetivo de estudo. Trata-se de uma aplicação Back-end baseada em microsserviços para envio de mensagens entre os usuários. 

## Bibliotecas/Tecnologias Utilizadas
- NodeJS
- TypeScript
- Sequelize
- MySQL
- Express
- JsonWebToken
- Bcrypt
- Dotenv
